## What I Build

### [M365 LangChain Agent](https://github.com/NeerajMohanty/m365-langchain-agent)
RAG agent for Microsoft 365 that answers employee questions from an internal knowledge base with source citations.

Built with Python, LangChain, Azure OpenAI, Azure AI Search, and CosmosDB. Deployed on Azure Container Apps with managed identity based authentication.

Includes architecture documentation, a Chainlit debug UI for testing and tuning, and Azure AI Foundry registration support for Microsoft 365 and Teams workflows.
